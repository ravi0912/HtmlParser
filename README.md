# HtmlParser
A sample Html Parser to extract Data from web Page : http://www.espncricinfo.com/australia-v-india-2015-16/engine/match/895809.html?innings=1;view=commentary
And Storing it in Database(PhpMyAdmin) and used trigger to feed data in other table.

Language Used : Python
Database Used : Mysql
Feature : Html Parser, Trigger(Mysql) and Regex
